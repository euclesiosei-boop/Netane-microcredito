💰 Sobre o Sistema
O Netane Microcrédito EI é um sistema web completo de gestão de microcrédito, desenvolvido para pequenas empresas financeiras em Moçambique.

Funciona 100% no browser sem necessidade de servidor — todos os dados ficam guardados localmente em segurança.
